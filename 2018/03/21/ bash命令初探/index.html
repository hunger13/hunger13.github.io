bash命令初探

发表于 2018-03-21

| 命令 | 操作 |
| export | 显示所有的环境变量，如果你想获取某个变量的详细信息，使用 echo $VARIABLE_NAME |
| :-: | :-: |
| whereis | whereis使用系统自动构建的数据库来搜索可执行文件，源文件和手册页面 |
| which | 它在环境变量PATH指定的目录中搜索可执行文件。此命令将打印可执行文件的完整路径 |
| clear | 清除窗口上的内容 |
| ls | 列出您的文件。ls有很多选项：-l列出“长格式”的文件，其中包含文件的确切大小，拥有该文件的人员，有权查看该文件，以及何时进行上次修改。-a列出所有文件，包括隐藏文件 |
| touch | 创建或更新您的文件 |
| cat | 显示/复制/合并/创建新的文本文件 |
| mv | 将文件从一个位置移动到另一个位置 |
| cp | 将文件从一个位置复制到另一个位置 |
| rm | 删除文件。在目录上使用此命令会给您显示一个错误： rm: directory: is a directory。 为了删除目录，你必须传递-rf去递归删除目录中的所有内容。 |
| diff | 比较文件,并列出他们的差异 |
| chmod | 更改文件的读,写,执行权限 |
| gzip | 压缩文件 |
| gunzip | 解压缩gizp文件 |
| gzcat | 查看gzip文件,且不解压 |
| echo | 显示文字 |
| mkdir | 生成一个新的目录 |
| cd | 从一个目录转移到另外一个目录 |
| pwd | 获取当前所在目录 |
| ssh | 登录远程机器,此命令还接受-p可用于连接到特定端口的选项 |
| whoami | 返回当前登录用户名 |
| passwd | 允许当前登录用户更改密码 |
| quota | 显示磁盘配额 |
| date | 显示当前日期和时间 |
| uptime | 显示当前的正常运行时间 |
| man | 显示命令手册 |
| ps | 显示进程 |
| kill | 杀掉指定id进程 |
| top | 当前活动的进程 |
| ping | ping主机并输出结果 |
| whois | 获取域的whois信息 |
| dig | 获取域的DNS信息 |
| wget | 下载文件 |
