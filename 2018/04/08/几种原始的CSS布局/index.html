几种原始的CSS布局

发表于 2018-04-08

在flex和gird没有普及或适配不太好的时候,有很多奇葩但实际使用比较多的布局方式,下面来简单的整理一下:

两栏左右布局

没有任何追求最原始也是最简单的方式:浮动

还可以用负margin

三栏布局

三栏布局比较无脑的方式是单纯的利用定位来实现

还有风靡一时的圣杯布局以及双飞翼布局,因为整体思路差不多,我就只写双飞翼布局

水平居中

  1. 最古老的实现方案，在容器上定义一个宽度，然后配合margin的左右值为auto实现效果
  2. 容器设置inline-block,容器的父元素设置text-align:center
  3. 浮动+相对定位+left: 50%
  4. 绝对定位+left: 50%+margin-left: -(宽度值的一半)
  5. 块级元素 margin:auto

垂直居中

  1. 绝对定位+top: 50%+margin-top: -(高度值的一半)
  2. 在文本中line-height等于height
